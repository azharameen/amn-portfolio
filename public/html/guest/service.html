LiveFeeds

Load the below script file in to your page.

URL : http://amn-portfolio.herokuapp.com/js/livefeed.js

Cheat Code

Fist get your secret key and then connect to the livefeed service. Initialize it after page loaded completely.
var feed = new LiveFeed(secret key);

LiveFeed send data to all who are connected to your project. receive the data as 'livefeed_cb' callback. it won't save, only send and receive at the current time.

SendFeed send data to all who are connected to your project. receive the data as 'receive_feed' callback. it will be saved. can send and receive data at any time, data can be stored and used later. Note: Same key request will override the data.

SendMsg send data to a particular user who is connected to your project. receive the data as 'receive_msg' callback. it won't be saved. can send and receive data only current time, data can't be stored and nor used later. Note: User id should be passed so to send only to particular user.
